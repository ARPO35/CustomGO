# CustomGO

**用于 CS2 的Legit CFG**
[*仓库地址*](https://github.com/ARPO35/CustomGO/)

**使用前注意备份设置，此CFG会修改多项绑定和设定**

### 如何使用?
将 CSTM 文件夹放在`steam\steamapps\common\Counter-Strike Global Offensive\game\csgo\cfg\`目录下

在Steam启动项中添加 `+exec cstm\main`
